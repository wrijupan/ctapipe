CameraGeometry¶

class ctapipe.io.camera.CameraGeometry(cam_id, pix_id, pix_x, pix_y, pix_area, neighbors, pix_type)[source]¶

Bases: object

CameraGeometry is a class that stores information about a Cherenkov Camera that us useful for imaging algorithms and displays. It contains lists of pixel positions, areas, pixel shapes, as well as a neighbor (adjacency) list for each pixel.

The class is intended to be generic, and work with any Cherenkov Camera geometry, including those that have square vs hexagonal pixels, gaps between pixels, etc.

Parameters:

self: type

position of each pixel (y-coordinate)

pix_type: string

either ‘rectangular’ or ‘hexagonal’
